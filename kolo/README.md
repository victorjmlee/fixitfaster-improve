# kolo – PDF 키워드 추출 (견적용)

PDF에서 **연구**, **용역**, **원가** 키워드에 해당하는 내용만 뽑아 견적 형태로 출력합니다.

## 사용법

1. **PDF 넣기**  
   `kolo` 폴더에 PDF를 넣고 `sample.pdf` 또는 `upload.pdf` 로 저장하거나,  
   아래처럼 경로를 인자로 넘깁니다.

2. **실행**
   ```bash
   cd kolo
   npm run extract
   ```
   또는 PDF 경로 지정:
   ```bash
   node extract.mjs ./내문서.pdf
   ```

## 출력 형식

키워드별로 해당 문장/문단이 정리되어 견적처럼 출력됩니다.

- **연구** – "연구"가 포함된 내용
- **용역** – "용역"이 포함된 내용  
- **원가** – "원가"가 포함된 내용

## 키워드 변경

`extract.mjs` 상단의 `KEYWORDS` 배열을 수정하면 됩니다.

```js
const KEYWORDS = ["연구", "용역", "원가"];
```
